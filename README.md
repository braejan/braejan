## What I'm building

**witsaba** is a CPN engine where _every_ agent — a single LLM call, a tool invocation, a bash run, a ReAct loop, a sub-team — is the **same shape**: a Coloured Petri Net. A built-in **Architect** (itself a 7-place / 6-transition CPN) composes new CPNs from a library of primitives + operators to solve arbitrary tasks.

The thesis: agents don't need yet another framework. They need a **mathematical substrate** with verifiable boundedness, deterministic replay, typed budgets, and OTel GenAI semantics out of the box.

The metaphor is fungal. The Blueprint library _grows_ with use. The Architect _grows_ new CPNs from primitives. Strength is in the growth, not in any fixed configuration.
